# Atlas

Atlas is a personal reflection app designed with a calm, premium wellness interface.

## Overview

Atlas focuses on four core experiences:
- **Journal**: a timeline + editor workflow for daily reflections.
- **Ask**: a question-and-answer workspace with source-aware response UI.
- **Weekly**: a structured weekly reflection report with themes and experiments.
- **Settings**: privacy-focused controls and in-app guidance.

## Product Status

This repository represents the current MVP foundation:
- polished Next.js App Router frontend
- responsive layouts for desktop and mobile
- motion-enhanced interactions and glassy card system
- reusable component architecture for continued iteration

## Stack

- Next.js (TypeScript)
- Tailwind CSS
- shadcn-style UI primitives
- framer-motion
- lucide-react

## Notes

Atlas is currently positioned as a strong product baseline with room for future backend and data integrations.
